
1. Requires: 

Python 2.7

2. Package requirements:

- Matplotlib
- numpy 

3. Installing Matplotlib:

pip install matplotlib

4. Installing numpy:

pip install numpy

5. Running the system:

Put all the scipts in the same directory then run main.py however you typically run python code. 

Or, from the terminal: Use the cd command to change to the directory containing the python code and then type:

python main.py 

at the terminal.
